혜원 전신첩 NFT 프리세일 민팅 가이드

민팅 페이지로 가기

Go to mobile wallet install guide ( Metamask )

권장 브라우저: Chrome / Brave / Microsoft Edge

혜원 전신첩 NFT 민팅 화면은 권장 브라우저에 최적화 되어있습니다.


  • Step 1 : 우측 상단 Connect Wallet 버튼을 눌러 지갑을 연결해주세요.

    • Important : 메타마스트 지갑의 네트워크는 Ethereum Mainnet으로 설정합니다.

  • Step 2 : 지갑 연결 및 연결 할 지갑 주소를 선택하세요.
  • Step 3 : 화면 하단의 NFT Mint 버튼을 선택하세요.
  • Step 4 : 민팅할 수량을 선택하고 NFT Mint 버튼을 선택하세요.
    1회에 최대 20개까지 민팅할 수 있습니다.

    • Click NFT Mint!

  • Step 4 : Confirm Transaction

    • 중요 : 가스비가 높은 경우 - 민팅이 불가능하거나 민팅이 불가능한 주소일 경우(선판매의 경우) 가스가 평소보다 높아집니다.
    그럼에도 불구하고 민팅을 하면 거래가 실패하고 가스비는 소모됩니다..

  • Step 5 : 민팅이 완료될 때 까지 기다립니다.
  • Step 6 : 성공!
    민팅 직후에는 어떤 NFT 이미지를 소유하게 될 지 알 수 없습니다.
    판매 종료 이후 최대 3일 이내에 랜덤 방식으로 본인 소유의 NFT 이미지가 확정됩니다.
    확정 이전에는 OpenSea에 데모이미지가 보여지나 이후에는 구매자 소유의 NFT 이미지가 정상적으로 보여집니다.

Happy minting ~
